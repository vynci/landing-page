Technology Stack

I tackle software development using backend technologies such as node.js, docker, mysql/mariadb, mongodb, rabbitmq, and mosquitto(mqtt). I also use front-end technologies such as angular, backbone, vue.js in building browser applications, and ionic/cordova in building hybrid mobile applications.

In firmware and embedded systems, I develop using c++ and objective-c. I usually build hardware projects using microcontrollers such as Arduino, ESP8266, PIC, STM, and single-board computers such as RaspberryPi.

On my freetime I usually dabble with cryptocurrency and blockchain technologies. I'm currently doing experiments with decentralized applications using Ethereum Platfom, Smart Contracts, and IPFS.
